We are learning Git.

git 基本命令

  1. 全局配置
    • git config -global user.name ''
    • git config -global user.email ''
  2. 本地配置
    • git config -local user.name ''
    • git config -local user.email ''
  3. 初始化仓库
    • 已存本地项目，在根路径下： git init
    • 初始化空仓库：git init 仓库名
  4. 将文件添加到暂存区
    • 单个文件：git add 文件名/文件夹名
    • 所有文件：git add -u
  5. 提交变更
    • 提交暂存区变更： git commit -m "变更信息"
    • 直接提交，未存入暂存区：git commit -am "变更信息"
  6. 仓库状态：git status (提示哪些文件改变为暂存，哪些暂存未提交)
  7. 已提交文件重命名的简便方法：git mv readme readme.md
  8. 日志
    • 当前分支版本信息：git log
    • 查看简洁版本历史：git log --oneline
    • 最近几次提交版本：git log -n2 --oneline（2次）
    • 所有分支版本信息：git log --all
    • 所有分支图形化版本信息：git log --all --graph
    • 多个命令组合：git log --oneline --all --graph -n2
  9. 跳转到git log 的帮助文档网页:git help --web log
  10. 分支
    • 查看当前分支信息：git branch -v
    • 查看所有分支名字：git branch -a
    • 查看所有分支信息：git branch -av
    • 创建新分支,指向指定commit：git checkout -b [分支名字] [commit]
  11. Git对象彼此关系commit -> tree(文件夹）->blob（具体文件）
    查看git里面的对象的内容
    git cat-file -p [文件hash值]
  12. 查看git里面的存储对象的类型
    git cat-file -t [文件hash值]

mac 终端命令

  1. 清空屏幕：clear
  2. 当前文件路径：pwd
  3. 拷贝文件：cp 文件相对路径 存放的相对路径
  4. 改变当前目录：cd dirname进入下一层一路径 cd ../ 返回上一层 cd . 返回根路径
  5. 拷贝文件夹：cp -r 文件夹相对路径 存放的相对路径
  6. 创建文件夹：mkdir name
  7. 删除文件夹：rmkdir name
  8. 移动或重命名一个目录：mvdir dir1 dir2
  9. 在命令行打开当前文件 vi name
  10. 显示当前目录内容：ls -la
  11. 编辑使用vi打开的文件 ，输入i-变成可编辑状态
  12. vi编辑状态回到命令行：按esc 取消编辑 然后 :w 保存文件但不退出vi :w file 将修改另外保存到file中，不退出vi :w! 强制保存，不推出vi :wq 保存文件并退出vi :wq! 强制保存文件，并退出vi q: 不保存文件，退出vi :q! 不保存文件，强制退出vi :e! 放弃所有修改，从上次保存文件开始再编辑
  13. 改变文件名或移动文件： mv name1 name2 将name1重命名为name2 mv: dir1 dir2 将路经dir1下的文件移动到dir2下
  14. 创建文件和内容：echo 'hello world' >readme
  15. 查看.git/objects文件夹里面的文件类型
    find .git/objects -type f
  16. cat+文件名：将普通文件内容打印到屏幕上
