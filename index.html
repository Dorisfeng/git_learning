git 基本命令

  1. 全局配置 git config -global user.name '' git config -global user.email '' 本地配置 git config -local user.name '' git config -local user.email ''
  2. 初始化已存在仓库在文件目录下 git init 初始化空仓库 git init 仓库名
  3. 将文件添加到暂存区 git add 文件名/文件夹名 所有文件 git add -u
  4. 提交变更 git commit -m "变更信息"
  5. git状态 git status 提示哪些文件改变为暂存，哪些暂存未提交
  6. git 日志 git log

mac 终端命令

  1. 清空屏幕：clear
  2. 当前文件路径：pwd
  3. 拷贝文件：cp 文件相对路径 存放的相对路径
  4. 改变当前目录：cd dirname进入下一层一路径 cd ../ 返回上一层 cd . 返回根路径
  5. 拷贝文件夹：cp -r 文件夹相对路径 存放的相对路径
  6. 创建文件夹：mkdir name
  7. 删除文件夹：rmkdir name
  8. 移动或重命名一个目录：mvdir dir1 dir2
  9. 在命令行打开当前文件 vi name
  10. 显示当前目录内容：ls -la
  11. 编辑使用vi打开的文件 按键i
  12. vi编辑状态回到命令行：按esc 取消编辑 然后 shit+z(两次)
  13. 改变文件名或移动文件： mv name1 name2 将name1重命名为name2 mv: dir1 dir2 将路经dir1下的文件移动到dir2下
